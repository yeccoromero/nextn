# **App Name**: Vectoria

## Core Features:

- Shape Creation: Create basic shapes: rectangles, circles, polygons, and stars.
- Object Modification: Modify object properties such as color, size, and position.
- Multiple Selection: Select multiple objects for grouped operations.
- Anchor and Box: Manipulate objects using anchor points and bounding boxes.
- Text Addition and Formatting: Add and format text elements within the SVG design.
- SVG Export: Export designs in SVG format.

## Style Guidelines:

- Primary color: Cool cyan (#42cbf5) to provide a vibrant technical accent.
- Background color: Light gray (#f0f0f0), almost white, creating a clean, neutral canvas.
- Accent color: Darker gray (#666666) for UI controls to ensure clear affordances.
- Font: 'Inter', a grotesque sans-serif known for its clean, modern appearance, used throughout for both headings and body text.
